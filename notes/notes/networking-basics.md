 🌐 Networking Basics

   📌 IP Address
A unique identifier for a device on a network.

Example:
192.168.1.1

---

   🚪 Ports
Ports are like doors that allow services to run.

Common ports:
- 80 → HTTP
- 443 → HTTPS
- 22 → SSH

---

   🌍 DNS
DNS converts domain names into IP addresses.

Example:
google.com → IP address

---

   🧠 What I Learned
- Every device has an IP
- Ports control communication
- DNS helps access websites easily
