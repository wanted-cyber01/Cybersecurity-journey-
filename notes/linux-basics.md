 🐧 Linux Basics

   📌 What is Linux?
Linux is an operating system used by developers and cybersecurity professionals. It gives more control over the system compared to Windows.

---

  💻 Basic Commands

- ls → list files in directory  
- cd → change directory  
- pwd → show current directory  
- mkdir → create folder  
- rm → delete file  
- clear → clear terminal
