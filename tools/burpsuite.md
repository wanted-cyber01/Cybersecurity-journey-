 🛠️ Burp Suite

   📌 What is Burp Suite?
Burp Suite is a tool used to test web applications by intercepting and modifying HTTP requests.

---

   ⚙️ What it Does
- Intercepts requests
- Allows modification of data
- Helps find vulnerabilities

---

   🧠 What I Learned
- Websites send requests and responses
- Burp can capture and edit them
- Useful for web security testing
